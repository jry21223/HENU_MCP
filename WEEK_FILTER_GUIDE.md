# 本周课表过滤功能使用指南

## 功能说明

新增的 `latest_schedule_current_week` 工具可以根据当前周次自动过滤课程，只显示本周真正在上的课。

### 为什么需要这个功能？

课表中的时间信息格式如 `"1-18[3-4]"` 表示：
- `1-18`: 课程在第1-18周上课
- `[3-4]`: 在第3-4节上课

如果当前是第10周，而某门课只在"1-9周"上课，那么这门课本周实际上没有，不应该显示在课表中。

## 使用方法

### 1. 配置学期开始日期

编辑 `semester_config.json` 文件：

```json
{
  "semester_start": "2026-03-03",
  "semester_name": "2025-2026学年第二学期",
  "note": "semester_start是学期第1周的周一日期"
}
```

**重要**: `semester_start` 必须是第1周的周一日期，不是学期第一天！

例如：
- 如果学期3月5日（周三）开始
- 那么第1周的周一是3月3日
- 所以 `semester_start` 应该设置为 `"2026-03-03"`

### 2. 使用MCP工具

#### 方式1：获取本周课表（推荐）

```
调用 latest_schedule_current_week 工具
```

返回结果会包含：
- `current_week`: 当前周次
- `schedule`: 过滤后的课表（只包含本周的课）
- 每门课会额外包含：
  - `current_week`: 当前周次
  - `weeks`: 这门课在哪些周上课
  - `total_weeks`: 总共上几周

#### 方式2：获取完整课表

```
调用 latest_schedule 工具
```

返回所有课程，不进行周次过滤。

## 示例

### 输入
当前是第2周，查询本周课表

### 输出
```json
{
  "success": true,
  "current_week": 2,
  "schedule": {
    "星期一": [
      {
        "course": "高等数学A（二）",
        "teacher": "裴振磊",
        "time": "1-18[1-2]",
        "location": "十号楼220",
        "current_week": 2,
        "weeks": [1, 2, 3, ..., 18],
        "total_weeks": 18
      }
    ]
  },
  "msg": "当前第2周的课表"
}
```

### 过滤效果

假设有以下课程：
- 课程A: "1-9[1-2]" - 只在1-9周上课
- 课程B: "1-18[3-4]" - 在1-18周上课
- 课程C: "10-18[6-7]" - 只在10-18周上课

当前第2周时：
- ✅ 课程A 会显示（2在1-9范围内）
- ✅ 课程B 会显示（2在1-18范围内）
- ❌ 课程C 不会显示（2不在10-18范围内）

当前第10周时：
- ❌ 课程A 不会显示（10不在1-9范围内）
- ✅ 课程B 会显示（10在1-18范围内）
- ✅ 课程C 会显示（10在10-18范围内）

## 支持的时间格式

工具可以解析以下格式：

1. **连续周次**: `"1-18[3-4]"` - 第1到18周
2. **多段周次**: `"1-9,11-18[1-2]"` - 第1-9周和第11-18周（跳过第10周）
3. **单独周次**: `"1,3,5,7[6-7]"` - 第1、3、5、7周（单双周）

## 常见问题

### Q: 如何确定学期开始日期？

A: 找到学期第一周的周一日期。例如：
- 学期3月5日（周三）开始
- 往前推到周一：3月3日
- 设置 `semester_start` 为 `"2026-03-03"`

### Q: 周次计算不准确怎么办？

A: 检查 `semester_config.json` 中的日期是否正确。可以手动计算：
```
当前周次 = (今天 - 学期开始日期) / 7 + 1
```

### Q: 某些课程没有周次信息怎么办？

A: 如果课程的 `time` 字段为空或无法解析，该课程会被保留（不过滤）。

### Q: 可以查看其他周的课表吗？

A: 目前只支持查看当前周。如果需要查看其他周，可以临时修改 `semester_config.json` 中的日期。

## 技术细节

### 周次计算公式

```python
start_date = datetime.strptime(semester_start, "%Y-%m-%d").date()
now = datetime.now().date()
days_diff = (now - start_date).days
current_week = (days_diff // 7) + 1
```

### 时间字符串解析

使用正则表达式 `r'(.+?)\[(.+?)\]'` 提取周次和节次：
- 周次部分支持 `-` 和 `,` 分隔
- 节次部分支持 `-` 表示范围
